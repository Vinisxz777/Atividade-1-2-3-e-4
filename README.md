# Atividade 3

## Descrição
Dashboard interativo de análise de salários utilizando Streamlit.

## Funcionalidades
- Filtro por categoria de salário
- Gráfico interativo com Plotly
- Upload de CSV
- Tabela de dados filtrados

## Tecnologias
- Python
- Streamlit
- Pandas
- Plotly

## Como executar
pip install streamlit pandas plotly

streamlit run app.py
